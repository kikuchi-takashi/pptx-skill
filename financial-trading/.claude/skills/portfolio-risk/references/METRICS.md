# リスク指標の定義

`scripts/risk_metrics.py` が算出する指標。

| 指標 | 定義 | 読み方 |
|---|---|---|
| 時価評価額 / 損益 | quantity × 直近終値、(直近終値 - cost_basis) × quantity | 含み損益の把握 |
| 構成比 | 各銘柄の時価評価額 ÷ ポートフォリオ合計 | 配分の把握 |
| HHI（ハーフィンダール指数） | Σ(構成比^2)、0〜1（1に近いほど集中） | 銘柄集中度。0.25超は「上位銘柄への集中が強い」目安 |
| 年率ボラティリティ | ポートフォリオ日次リターンの標準偏差 × √252 | ポートフォリオ全体の値動きの荒さ |
| 歴史的VaR(95%) | 過去の日次リターン分布の下位5%点 × 時価評価額（マイナス値=評価損方向） | 「よくある悪い日」でどの程度の評価損が起こりうるかの目安（正規分布を仮定しない） |
| 最大ドローダウン | 期間中のピークからの最大下落率 | 過去に実際に起きた最大の目減り幅 |
| シャープレシオ（簡易） | (年率リターン - 無リスク金利) ÷ 年率ボラティリティ、無リスク金利は既定0% | リスクあたりのリターン効率の目安 |
| 相関行列 | 保有銘柄間の日次リターンの相関係数 | 分散効果の有無（相関が高い銘柄同士は分散効果が薄い） |

## 数値の限界（レポートに明記すること）

- VaR・ボラティリティは過去データに基づく統計量であり、将来の下落を保証・予測するもの
  ではない。特に歴史的VaRは参照期間に依存する（期間中に大暴落が含まれるかどうかで
  大きく変わる）。
- シャープレシオは簡易計算（無リスク金利=0%が既定）であり、正確な比較には
  各国の無リスク金利を反映する必要がある。
- 通貨が混在するポートフォリオでは、為替変動リスクは上記指標に別途含まれていない
  （個別に為替レートの変動リスクを検討する必要がある旨を明記する）。

## リバランス案の出し方

- ユーザーが目標配分（例: 「株60%/現金40%」「各銘柄均等」）を指定していれば、それに
  対する現状の乖離を銘柄別に算出し、「増減させる金額・数量の目安」を提示する。
- 目標配分の指定がない場合、機械的な既定（例: 均等配分）を使ったことを明記した上で、
  あくまで参考値として提示する——「これに従うべき」という助言にはしない。
- リバランス案は数量の提示までとし、実際の発注は行わない。発注案の作成が必要なら
  `order-ticket-draft` スキルに引き継ぐ。
