# 既定で計算する指標

`scripts/make_report.py` がデフォルトで計算する指標一覧。特定の指標だけ求められた
場合はそれだけを計算・掲載してよい（全部を毎回出す必要はない）。

| 指標 | 計算 | 用途 |
|---|---|---|
| SMA25 / SMA75 | 単純移動平均（25日・75日） | トレンド方向の把握、ゴールデン/デッドクロスの確認 |
| RSI(14) | 14日相対力指数（本実装は単純移動平均ベースの近似式であり、証券会社の
  チャートツール等で一般的なWilderの平滑化とは値がわずかに異なる場合がある） | 過熱・売られ過ぎの目安（70超/30未満を参考値として明記） |
| ボリンジャーバンド | SMA20 ± 2σ | 価格の変動レンジの目安 |
| 騰落率 | 期間内の変化率、前日比 | 値動きの大きさの把握 |
| 出来高 | 日次出来高（棒グラフ） | 値動きの裏付けとなる商いの厚み |
| 年率ボラティリティ | 直近20日の日次リターンの標準偏差 × √252（移動窓） | 値動きの荒さの定量化 |

## レポートに書く際の注意

- 指標は「事実の記述」として書く。「RSIが70を超えている＝過熱感がある」という
  一般的な解釈の説明は良いが、「だから今売るべき」のような断定的な投資判断は書かない。
- 複数銘柄比較では、単位・スケールを揃える（価格そのものではなく騰落率で比較する等）。
- 指標名の初出時に日本語の一言説明を添える（受け手が個人トレーダーで前提知識に幅がある）。
